# 記帳本（單人版）

一個離線可用、資料存進**你自己的 Google Sheets** 的個人記帳 PWA。單一 HTML 檔，不需要伺服器、不需要安裝。

- **單人**：只記你一個人的帳（沒有雙人比較、付款人切換）
- **預設臺幣 NT$**，幣別可切換（含 菲律賓披索 PHP、越南盾 VND、印尼盾 IDR、泰銖、美金、日圓…）
- **沒有語音功能**（純手動輸入，含內建小算盤 `＋－×÷`）
- 功能：今日 / 本月總覽、月預算、自訂期間查詢（圓餅圖 / 樹狀圖 / 每日趨勢）、定期支出、CSV 匯出、中英文切換
- 匯率可一鍵從台灣銀行更新

## 檔案

| 檔案 | 用途 |
|------|------|
| `index.html` | App 本體，用瀏覽器打開就能用 |
| `Code.gs` | Google Apps Script 後端（貼到你的 Google Sheet 用） |
| `AGENTS.md` | 給 Codex Desktop 看的架設指引 |

## 怎麼用（兩種路）

### A. 讓 Codex Desktop 帶你架（推薦）
用 Codex Desktop 打開這個資料夾，直接說「**幫我照 AGENTS.md 把這個記帳本接上我自己的 Google Sheet**」，它會一步步帶你做完下面 B 的所有步驟。

### B. 自己手動架（約 5 分鐘）
1. 開 [sheets.new](https://sheets.new) 建一張新試算表，隨意命名（例：我的記帳本）。
2. 在那張試算表裡：**擴充功能 → Apps Script**。
3. 把 `Code.gs` 全部內容貼進去，蓋掉預設的 `myFunction`，按 **儲存**。
4. 右上角 **部署 → 新增部署作業**：
   - 類型選 **網頁應用程式 (Web app)**
   - 「執行身分」選 **我**
   - 「誰可以存取」選 **任何人 (Anyone)**
   - 按 **部署**，第一次會要求授權 → 允許（出現「未驗證」時點「進階 → 前往…（不安全）」是正常的，因為是你自己的腳本）。
5. 複製產生的網址（結尾是 `/exec`）。
6. 用瀏覽器打開 `index.html` → 第一次會跳出設定視窗 → 貼上那個 `/exec` 網址 → 測試連線成功即可開始記帳。

> 想放到 iPhone 主畫面：用 Safari 開 `index.html`（建議先丟到雲端或自己的網頁空間），分享 → 加入主畫面。

## 換手機 / 換瀏覽器
資料都在你的 Google Sheet，換裝置只要再開一次 `index.html`、貼上同一個 `/exec` 網址，就會把雲端資料同步回來。本機只是快取。

## 備註
- 後端 `Code.gs` 沿用原本的結構（試算表仍保留 `user` 欄位，單人版固定填你的名字，不影響使用）。
- 預設匯率是估值，接上後到「設定」頁按「🔄 更新匯率」可抓台灣銀行即期匯率。
